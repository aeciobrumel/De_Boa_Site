## Instalação

Pré-requisitos: Node.js e npm.

1. Entre na pasta do app: `cd frontend`
2. Instale as dependências: `npm install`
3. Inicie o servidor de desenvolvimento: `npm run dev`

Scripts úteis:

- `npm run build`
- `npm run preview`
- `npm run lint`
- `npm run format`
